数学分析

定义和概念

  1. $\mathbb{N}$——自然数集，由计数时产生的数组成。
  2. $\mathbb{Z}$——整数集，由自然数、零以及与自然数相反的数组成。
  3. $\mathbb{Q}$——有理数集，由可以表示为$\frac{z}{n}$形式的数组成，其中$z\in\mathbb{Z}$，$n\in\mathbb{N}$。
  4. $\mathbb{I}$——无理数集，由不能表示为$\frac{z}{n}$形式的数组成，其中$z\in\mathbb{Z}$，$n\in\mathbb{N}$，例如$e$、$\pi$、$\sqrt{3}$等。
  5. $\mathbb{R}$——实数集，由有理数和无理数组成。
  6. $\overline{\mathbb{R}}$——扩展实数集，由实数加上$+\infty$和$-\infty$组成。
  7. 点$x$的邻域$U(x)$是指包含该点的任意区间。
  8. 点$x$的去心邻域$\stackrel{\circ}{U}(x)$是指不包含点$x$本身的邻域$U(x)$。
  9. （当$\varepsilon>0$时）点$x_0$的$\varepsilon$邻域是指区间$(x_0 - \varepsilon, x_0 + \varepsilon)$，即$U_{\varepsilon}(x_0)={x\in\mathbb{R}: x_0 - \varepsilon < x < x_0 + \varepsilon}$。
  10. 点$x_0$的右（右单侧）$\delta$邻域是指半区间$[x_0, x_0 + \delta)$，其中$\delta > 0$，即$U_{\delta}^{+}(x_0)={x\in\mathbb{R}: x_0 \leq x < x_0 + \delta}$，$\delta > 0$。
  11. 点$x_0$的左（左单侧）$\delta$邻域是指半区间$(x_0 - \delta, x_0]$，其中$\delta > 0$，即$U_{\delta}^{-}(x_0)={x\in\mathbb{R}: x_0 - \delta < x \leq x_0}$，$\delta > 0$。
  12. 点$+\infty$的邻域是指区间$(a, +\infty)$，其中$a > 0$，即$U(+\infty)={x\in\mathbb{R}: x > a}$，$a > 0$。
  13. 点$-\infty$的邻域是指区间$(-\infty, -a)$，其中$a > 0$，即$U(-\infty)={x\in\mathbb{R}: x < -a}$，$a > 0$。
  14. 点$\infty$（无符号无穷大）的邻域是指两个区间$(-\infty, -a)\cup(a, +\infty)$的并集，其中$a > 0$，即$U(\infty)={x\in\mathbb{R}: |x| > a}$，$a > 0$。
  15. 序列${X_n}$是指以自然数为自变量的数值函数。如果对于自然数$n$，有一个数$x_n$与之对应，那么这个数$x_n$就称为序列的第$n$项；$n$称为项$x_n$的序号。
  16. 数列${X_n}$称为非递减的，如果对于所有$n\in\mathbb{N}$，都有$x_{n + 1} \geq x_n$。
  17. 数列${X_n}$称为递增的，如果对于所有$n\in\mathbb{N}$，都有$x_{n + 1} > x_n$。
  18. 数列${X_n}$称为非递增的，如果对于所有$n\in\mathbb{N}$，都有$x_{n + 1} \leq x_n$。
  19. 数列${X_n}$称为递减的，如果对于所有$n\in\mathbb{N}$，都有$x_{n + 1} < x_n$。
  20. 非递减、非递增、递减和递增的数列统称为单调数列。
  21. 数列${X_n}$称为常数列，如果对于所有$n\in\mathbb{N}$，都有$x_n = c$，其中$c\in\mathbb{R}$。
  22. 数列${X_n}$称为有上界的，如果存在一个实数$M$，使得对于所有$n\in\mathbb{N}$，都有$x_n \leq M$。
  23. 数列${X_n}$称为有下界的，如果存在一个实数$M$，使得对于所有$n\in\mathbb{N}$，都有$x_n \geq M$。
  24. 数列${X_n}$，如果既有上界又有下界，则称为有界数列：存在$M > 0$，$M\in\mathbb{R}$，使得对于所有$n\in\mathbb{N}$，都有$|x_n| \leq M$。
  25. 数$a$称为数列${X_n}$的极限，如果对于任意小的正数$\varepsilon$，都存在一个依赖于$\varepsilon$的自然数$N$，使得对于所有$n > N$，不等式$|x_n - a| < \varepsilon$都成立，即$\lim_{n \to \infty} x_n = a \Leftrightarrow \forall \varepsilon > 0 \exists N = N(\varepsilon)\in\mathbb{N}: \forall n > N \Rightarrow |x_n - a| < \varepsilon$。
  26. 数值数列称为收敛数列，如果该数列存在极限，且极限是有限的。
  27. 数列${X_n}$称为基本数列（柯西数列），如果对于任意$\varepsilon > 0$，都存在一个自然数$N = N(\varepsilon)$，使得对于任意$m \geq N$和$n \geq N$，不等式$|x_m - x_n| < \varepsilon$都成立。
  28. （柯西定义）数$a$称为函数$f(x)$当$x \to x_0$时的极限，如果对于任意$\varepsilon > 0$，都存在一个正数$\delta = \delta(\varepsilon)$，使得对于任意$x \in \stackrel{\circ}{U}{\delta}(x_0)$，不等式$|f(x) - a| < \varepsilon$都成立，即$\lim_{x \to x_0} f(x) = a \Leftrightarrow \forall \varepsilon > 0 \exists \delta = \delta(\varepsilon) > 0: \forall x \in \stackrel{\circ}{U}{\delta}(x_0) \Rightarrow |f(x) - a| < \varepsilon$。
  29. （海涅定义）数$a$称为函数$f(x)$当$x \to x_0$时的极限，如果对于任何来自$\stackrel{\circ}{U}(x_0)$的点列${X_n}$，且$\lim_{n \to \infty} x_n = x_0$，都有$\lim_{n \to \infty} f(x_n) = a$成立，即$\lim_{x \to x_0} f(x) = a \Leftrightarrow {\forall x_n \in \stackrel{\circ}{U}(x_0), n\in\mathbb{N}} \cap \lim_{n \to \infty} x_n = x_0: \lim_{n \to \infty} {f(x_n)} = a$。
  30. 数$a$称为函数$f(x)$当$x \to x_0^+$时的右（右单侧）极限，如果对于任意$\varepsilon > 0$，都存在$\delta = \delta(\varepsilon) > 0$，使得对于任意$x \in \stackrel{\circ}{U}{\delta}^{+}(x_0)$（即$x_0 < x < x_0 + \delta$），不等式$|f(x) - a| < \varepsilon$都成立，即$\lim_{x \to x_0^+} f(x) = a \Leftrightarrow \forall \varepsilon > 0 \exists \delta = \delta(\varepsilon) > 0: \forall x \in \stackrel{\circ}{U}{\delta}^{+}(x_0) \Rightarrow |f(x) - a| < \varepsilon$。
  31. 数$a$称为函数$f(x)$当$x \to x_0^-$时的左（左单侧）极限，如果对于任意$\varepsilon > 0$，都存在$\delta = \delta(\varepsilon) > 0$，使得对于任意$x \in U_{\delta}^{-}(x_0)$（即$x_0 - \delta < x < x_0$），不等式$|f(x) - a| < \varepsilon$都成立，即$\lim_{x \to x_0^-} f(x) = a \Leftrightarrow \forall \varepsilon > 0 \exists \delta = \delta(\varepsilon) > 0: \forall x \in U_{\delta}^{-}(x_0) \Rightarrow |f(x) - a| < \varepsilon$。
  32. 函数$f(x)$在集合$D$上称为有界的，如果存在一个数$M > 0$，使得对于任意$x \in D$，不等式$|f(x)| \leq M$都成立。
  33. 函数$f(x)$（在定义域$D_f$上）称为有界的，如果存在一个数$M > 0$，使得对于任意$x \in D_f$，不等式$|f(x)| \leq M$都成立。
  34. 函数$f(x)$在点$a$的邻域内称为局部有界的，如果存在一个数$M > 0$和一个邻域$\stackrel{\circ}{U}{\delta}(a)$，使得对于任意$x \in \stackrel{\circ}{U}{\delta}(a)$，不等式$|f(x)| \leq M$都成立。
  35. 函数$f(x)$当$x \to x_0$（$x_0 \in \overline{\mathbb{R}}$）时称为无穷小量，如果$\lim_{x \to x_0} f(x) = 0$。
  36. 函数$f(x)$当$x \to x_0$（$x_0 \in \overline{\mathbb{R}}$）时称为无穷大量，如果$\lim_{x \to x_0} f(x) = \infty$。
  37. $\lim_{x \to 0} \frac{\sin(x)}{x} = 1$，这是第一个重要极限。
  38. $\lim_{x \to \infty} (1 + \frac{1}{x})^x = e$，这是第二个重要极限。
  39. 函数$\alpha(x)$和$\beta(x)$当$x \to x_0$时称为可比无穷小量，如果$\lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)}$或$\lim_{x \to x_0} \frac{\beta(x)}{\alpha(x)}$至少有一个极限存在。
  40. 函数$\alpha(x)$和$\beta(x)$当$x \to x_0$时称为不可比无穷小量，如果当$x \to x_0$时，它们的比值既不存在有限极限，也不存在无穷极限。
  41. 函数$\alpha(x)$和$\beta(x)$当$x \to x_0$时称为同阶无穷小量，并记为$\alpha(x) = O(\beta(x))$，如果当$x \to x_0$时，$\alpha(x) / \beta(x)$的极限存在且为非零有限数，即$\alpha(x) = O(\beta(x))$当$x \to x_0 \Leftrightarrow \exists \lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)} = c \in \mathbb{R} \backslash {0}$。
  42. 函数$\alpha(x)$当$x \to x_0$时称为比$\beta(x)$高阶的无穷小量，并记为$\alpha(x) = o(\beta(x))$，如果当$x \to x_0$时，$\alpha(x) / \beta(x)$的极限存在且等于零，即$\alpha(x) = o(\beta(x))$当$x \to x_0 \Leftrightarrow \exists \lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)} = 0$。
  43. 函数$\alpha(x)$当$x \to x_0$时称为比$\beta(x)$低阶的无穷小量，如果当$x \to x_0$时，$\alpha(x) / \beta(x)$的极限等于无穷大。
  44. 函数$\alpha(x)$和$\beta(x)$当$x \to x_0$时称为等价无穷小量，如果当$x \to x_0$时，它们的比值的极限等于$1$，即$\alpha(x) \sim \beta(x)$当$x \to x_0 \Leftrightarrow \lim_{x \to x_0} \frac{\alpha(x)}{\beta(x)} = 1$。
  45. 函数$\alpha(x)$当$x \to x_0$时称为相对于$\beta(x)$的$k$阶无穷小量，数$k(k > 0)$称为$\alpha(x)$相对于$\beta(x)$当$x \to x_0$时的无穷小阶数，如果函数$\alpha(x)$和$\beta^k(x)$当$x \to x_0$时是同阶无穷小量，即$\lim_{x \to x_0} \frac{\alpha(x)}{\beta^k(x)} = c \in \mathbb{R} \backslash {0}$。
  46. 函数$u(x)$当$x \to x_0$时称为相对于$w(x)$的$k$阶无穷大量，数$k(k > 0)$称为$u(x)$相对于$w(x)$当$x \to x_0$时的无穷大阶数，如果函数$u(x)$和$w^k(x)$当$x \to x_0$时是同阶无穷大量，即$\lim_{x \to x_0} \frac{u(x)}{w^k(x)} = c \in \mathbb{R} \backslash {0}$。
  47. 无穷小函数和的主部是指比其他每一项都低阶的无穷小量。
  48. 函数在点$x_0$处自变量的增量是指自变量从$x_0$变化到另一个值$x$时的变化量，即$\Delta x = x - x_0$。
  49. 函数在点$x_0$处的增量是指$\Delta y = f(x_0 + \Delta x) - f(x_0)$。
  50. （定义1）函数$f(x)$在点$x_0$处称为连续的，如果函数在该点存在有限极限，且该极限与函数在该点的值相等，即$\exists \lim_{x \to x_0} f(x) = f(x_0)$。
  51. （定义2）函数$f(x)$在点$x_0$处称为连续的，如果当自变量的增量$\Delta x \to 0$时，函数在该点的增量是一个无穷小函数。
  52. 函数$f(x)$在点$x_0$处称为右连续的，如果函数在该点存在有限的右极限，且该极限与函数在该点的值相等，即$\exists \lim_{x \to x_0^+} f(x) = f(x_0)$。
  53. 函数$f(x)$在点$x_0$处称为左连续的，如果函数在该点存在有限的左极限，且该极限与函数在该点的值相等，即$\exists \lim_{x \to x_0^-} f(x) = f(x_0)$。
  54. 函数$f(x)$在区间$(a, b)$上连续，如果它在该区间的每一点都连续。
  55. 函数$f(x)$在闭区间$[a, b]$上连续，如果它在区间$(a, b)$上连续，在点$a$处右连续，即$\lim_{x \to a^+} f(x) = f(a)$，在点$b$处左连续，即$\lim_{x \to b^-} f(x) = f(b)$。
  56. 如果给定的函数$f(x)$在点$x_0$处不连续，那么$x_0$称为函数$f(x)$的间断点。
  57. 第一类间断点是指函数的间断点，在该点函数的两个单侧极限都存在且有限。
  58. 第二类间断点是指函数的间断点，在该点函数的至少一个单侧极限不存在（特别地，等于无穷大）。
  59. 如果$x_0$是函数的第一类间断点，且函数在该点的两个单侧极限相等，但不等于函数在该点的值，或者函数在该点未定义，那么这样的间断称为可去间断点，$x_0$称为第一类可去间断点。
  60. 如果$x_0$是函数的第一类间断点，且函数在该点的单侧极限不相等，那么这样的间断称为不可去间断点，$x_0$称为第一类不可去间断点。
  61. 如果函数$f(x)$在点$x_0$处的极限$\lim_{\Delta x \to 0}\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$存在，那么这个极限称为函数$f(x)$在点$x_0$处的导数 。
  62. 如果$f(x)$在点$x_0$的右邻域有定义，并且$\exists \lim_{\Delta x \to 0^+}\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$，那么这个极限称为函数$f(x)$在点$x_0$处的右导数，记作$f_+’(x_0)$。
  63. 如果$f(x)$在点$x_0$的左邻域有定义，并且$\exists \lim_{\Delta x \to 0^-}\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$，那么这个极限称为函数$f(x)$在点$x_0$处的左导数，记作$f_-‘(x_0)$。
  64. 设函数$y = f(x)$在点$x_0$的某个邻域内有定义。如果函数在点$x_0$处的增量$\Delta y$可以表示为$\Delta y = A\Delta x + \alpha(\Delta x)\Delta x$的形式，其中$A$是一个与$\Delta x$无关的数，且$\lim_{\Delta x \to 0}\alpha(\Delta x) = 0$，那么函数在点$x_0$处可微。
  65. 函数$f(x)$在点$x_0$处的微分是指函数增量的线性主部，它与自变量的增量$\Delta x$有关。
  66. $n$阶微分是指$n - 1$阶微分的微分，即$d^n y = d(d^{n - 1}y) = f^{(n)}(x)dx^n$。
  67. 函数$y = f(x)$的$n$阶导数是指$n - 1$阶导数的导数，即$f^{(n)} = (f^{n - 1}(x))’$。
  68. 函数$f(x)$在区间$(a, b)$上称为递增的，如果对于任意的$x_1$，$x_2 \in (a, b)$，当$x_2 > x_1$时，不等式$f(x_2) > f(x_1)$成立。
  69. 函数$f(x)$在区间$(a, b)$上称为不增的，如果对于任意的$x_1$，$x_2 \in (a, b)$，当$x_2 > x_1$时，不等式$f(x_2) \leq f(x_1)$成立。
  70. 函数$f(x)$在区间$(a, b)$上称为递减的，如果对于任意的$x_1$，$x_2 \in (a, b)$，当$x_2 > x_1$时，不等式$f(x_2) < f(x_1)$成立。
  71. 函数$f(x)$在区间$(a, b)$上称为不减的，如果对于任意的$x_1$，$x_2 \in (a, b)$，当$x_2 > x_1$时，不等式$f(x_2) \geq f(x_1)$成立。
  72. 不减、不增、递减和递增的函数统称为单调函数。
  73. 函数$f(x)$称为严格单调的，如果它是递增的或递减的。
  74. 点$x_0$称为函数$f(x)$的局部极小值点，如果存在$U_{\delta}(x_0)$，使得对于任意的$x \in U_{\delta}(x_0)$，都有$f(x_0) \leq f(x)$。
  75. 点$x_0$称为函数$f(x)$的局部极大值点，如果存在$U_{\delta}(x_0)$，使得对于任意的$x \in U_{\delta}(x_0)$，都有$f(x_0) \geq f(x)$。
  76. 点$x_0$称为函数$f(x)$的严格局部极小值点，如果存在$\stackrel{\circ}{U}{\delta}(x_0)$，使得对于任意的$x \in \stackrel{\circ}{U}{\delta}(x_0)$，都有$f(x_0) < f(x)$。
  77. 点$x_0$称为函数$f(x)$的严格局部极大值点，如果存在$\stackrel{\circ}{U}{\delta}(x_0)$，使得对于任意的$x \in \stackrel{\circ}{U}{\delta}(x_0)$，都有$f(x_0) > f(x)$。
  78. 局部极大值点、严格局部极大值点、局部极小值点和严格局部极小值点统称为局部极值点。
  79. 严格局部极大值点和严格局部极小值点统称为严格局部极值点。
  80. 函数$f(x)$定义域内的点$x_0$称为临界点，如果函数在该点的导数为$0$或根本不存在。
  81. 函数$f(x)$定义域内的点$x_0$称为驻点，如果$f’(x_0) = 0$。
  82. 直线$Ax + By + C = 0$称为函数$y = f(x)$图像的渐近线，如果当点$M(x, f(x))$离坐标原点无限远时，该点到这条直线的距离趋于$0$。
  83. 直线$x = a$称为函数$y = f(x)$图像的垂直渐近线，如果至少有一个极限$\lim_{x \to a^ + (-)}f(x) = \infty$。
  84. 直线$y = kx + b$称为函数$y = f(x)$图像的右斜渐近线，如果当$x \to +\infty$时，函数可以表示为$f(x) = kx + b + \alpha(x)$的形式，其中$k$，$b \in \mathbb{R}$，$\alpha(x)$是无穷小函数。
  85. 直线$y = kx + b$称为函数$y = f(x)$图像的左斜渐近线，如果当$x \to -\infty$时，函数可以表示为$f(x) = kx + b + \alpha(x)$的形式，其中$k$，$b \in \mathbb{R}$，$\alpha(x)$是无穷小函数。
  86. 设函数$f(x)$在区间$(a, b)$上可微。如果函数$y = f(x)$的图像在区间$(a, b)$上位于其任意一条切线的下方，那么该图像在区间$(a, b)$上是上凸的。
  87. 设函数$f(x)$在区间$(a, b)$上可微。如果函数$y = f(x)$的图像在区间$(a, b)$上位于其任意一条切线的上方，那么该图像在区间$(a, b)$上是下凸的。
  88. 点$x_0 \in (a, b)$称为函数$f(x)$的拐点，如果函数在点$x_0$处连续，并且存在$\delta > 0$，使得函数$f(x)$在区间$(x_0 - \delta; x_0)$和$(x_0; x_0 + \delta)$上的凸性方向不同。

考试复习问题

1. 收敛数列极限的唯一性定理

使用列表中的定义15、25、26

如果一个数列有极限，那么这个极限是唯一的。

证明（反证法）：

假设$a, b \in \mathbb{R}$，$a \neq b$，且$a$和$b$是收敛数列${X_n}$的极限，即$\lim_{n \to \infty}x_n = a$，$\lim_{n \to \infty}x_n = b$，$a \neq b$。根据极限的定义，$\forall \varepsilon > 0$，$\exists N_1 = N_1(\varepsilon) \in \mathbb{N}$，使得$\forall n > N_1 \Rightarrow |x_n - a| < \varepsilon$；$\forall \varepsilon > 0$，$\exists N_2 = N_2(\varepsilon) \in \mathbb{N}$，使得$\forall n > N_2 \Rightarrow |x_n - b| < \varepsilon$。取$\varepsilon = \frac{|b - a|}{3}$，当$n > \max(N_1, N_2)$时，可得$|b - a| = |x_n - a + b - x_n| \leq |x_n - a| + |b - x_n| = |x_n - a| + |x_n - b| \Rightarrow |b - a| < 2\varepsilon$，即$|b - a| < 2 \cdot \frac{|b - a|}{3}$，也就是$\frac{1}{3}|b - a| < 0$，这是不可能的，所以$a \neq b$不成立，即$a = b$，极限是唯一的。

定理得证。

9. 第一个重要极限的推导：$\lim_{x \to 0}\frac{\sin(x)}{x} = 1$

设$0 < x < \frac{\pi}{2}$。考虑以坐标原点为圆心，半径为$R$的圆，它与$x$轴相交于点$A$、$\Pi$，设$\angle AOB = x$（弧度）。

设$CA$是垂直于该轴的垂线，$C$是线段$OB$延长线与该垂线的交点。那么$S_{\triangle OAB} < S_{扇形OAB} < S_{\triangle OAC}$，即$\frac{1}{2}R^{2}\sin(x) < \frac{1}{2}R^{2}x < \frac{1}{2}R^{2}\tan(x)$，

所以$\sin(x) < x < \tan(x)$，进而$1 < \frac{x}{\sin(x)} < \frac{1}{\cos(x)}$，也就是$1 > \frac{\sin(x)}{x} > \cos(x)$，当$x \in (0, \frac{\pi}{2})$。

考虑$x \in (-\frac{\pi}{2}, 0)$，作代换$\beta = -x$，则$\beta \in (0, \frac{\pi}{2})$，所以有$1 > \frac{\sin(\beta)}{\beta} > \cos(\beta)$。

再回到代换$\beta = -x$，得到$1 > \frac{\sin(-x)}{-x} > \cos(-x)$，即$1 > \frac{-\sin(x)}{-x} > \cos(x)$，也就是$1 > \frac{\sin(x)}{x} > \cos(x)$，当$x \in (-\frac{\pi}{2}, 0)$。

这样得到的不等式对于$x \in (-\frac{\pi}{2}, 0) \cup (0, \frac{\pi}{2})$都成立。

当$x \to 0$时取极限：$\left.\begin{array}{l} \lim_{x \to 0}\cos(x) = 1 \ \lim_{x \to 0}1 = 1 \end{array}\right} \Rightarrow$（根据定义）$\lim_{x \to 0}\frac{\sin(x)}{x} = 1$。

19. 函数在一点连续。基本函数的连续性定理

  • （定义1）函数$f(x)$在点$x_0$处称为连续的，如果在该点函数存在有限极限，且该极限与函数在该点的值相等，即$\exists \lim_{x \to x_0}f(x)=f(x_0)$。
  • （定义2）函数$f(x)$在点$x_0$处称为连续的，如果函数在该点的增量是当自变量增量趋于$0$（$\Delta x \to 0$）时的无穷小函数。

基本函数的连续性定理：所有基本函数在其定义域内都是连续的。

证明（以$y = \sin(x)$和$y = \cos(x)$为例）：

  • 考虑$y = \sin(x)$：

    求函数的增量$\Delta y = f(x+\Delta x)-f(x)=\sin(x+\Delta x)-\sin(x)=2\sin(\frac{x+\Delta x - x}{2})\cdot\cos(\frac{x+\Delta x + x}{2}) = 2\sin(\frac{\Delta x}{2})\cdot\cos(x+\frac{\Delta x}{2})$。

    那么$\lim_{\Delta x \to 0}\Delta y=\lim_{\Delta x \to 0}(2\sin(\frac{\Delta x}{2})\cdot\cos(x+\frac{\Delta x}{2})) = 0$，即$\lim_{\Delta x \to 0}\Delta y = 0$，根据定义$y = \sin(x)$在整个数轴上连续。

  • 考虑$y = \cos(x)$：

    求函数的增量$\Delta y = f(x+\Delta x)-f(x)=\cos(x+\Delta x)-\cos(x)= - 2\sin(x+\frac{\Delta x}{2})\cdot\sin(\frac{\Delta x}{2})$。

    那么$\lim_{\Delta x \to 0}\Delta y=\lim_{\Delta x \to 0}(-2\sin(x+\frac{\Delta x}{2})\cdot\sin(\frac{\Delta x}{2})) = 0$，即$\lim_{\Delta x \to 0}\Delta y = 0$，根据定义$y = \cos(x)$在整个数轴上连续。

20. 闭区间上连续函数的性质

  • 魏尔斯特拉斯第一定理：如果函数$y = f(x)$在$[a, b]$上连续，那么它在这个闭区间上有界。
  • 魏尔斯特拉斯第二定理：如果函数$y = f(x)$在$[a, b]$上连续，那么它在这个闭区间上有最大值和最小值。
  • 波尔查诺 - 柯西第一定理：如果函数$y = f(x)$在$[a, b]$上连续，且在这个闭区间的端点处取不同符号的值，即$f(a)\cdot f(b)<0$，那么至少存在一个点$c \in (a, b)$，使得函数值$f(c)=0$。
  • 波尔查诺 - 柯西第二定理：如果函数$y = f(x)$在$[a, b]$上连续且$f(a) \neq f(b)$，那么存在这样一个点$c \in (a, b)$，使得$f(a)<f(c)<f(b)$ 。
  • 反函数的连续性定理：如果函数$y = f(x)$在$[a, b]$上连续且单调递增（递减），那么在区间$[f(a), f(b)]$上存在且定义了反函数$x = f^{-1}(y)$，该反函数在这个区间上连续且递增（递减）。

21. 函数的间断点及其分类。示例

如果给定的函数$f(x)$在点$x_0$处不连续，那么$x_0$称为函数$f(x)$的间断点。

第一类间断点：函数的间断点中，函数在该点的两个单侧极限都存在且有限的点称为第一类间断点。

例如：$f(x)=\frac{\sin(x)}{x}$。考虑在点$x = 0$处的单侧极限和函数值：$\left.\begin{array}{l} \exists \lim_{x \to 0^+}\frac{\sin(x)}{x}=1 \ \exists \lim_{x \to 0^-}\frac{\sin(x)}{x}=1 \ x = 0 \notin D_f，即\nexists f(0) \end{array}\right} \Rightarrow$点$x = 0$是第一类间断点。

第二类间断点：函数的间断点中，函数在该点至少有一个单侧极限不存在（特别地，等于无穷大）的点称为第二类间断点。

例如：$f(x)=\frac{1}{x}$。考虑在点$x = 0$处的单侧极限和函数值：$\left.\begin{array}{l} \exists \lim_{x \to 0^+}\frac{1}{x}=+\infty \ \exists \lim_{x \to 0^-}\frac{1}{x}=-\infty \ x = 0 \notin D_f，即\nexists f(0) \end{array}\right} \Rightarrow$点$x = 0$是第二类间断点。

可去间断点：如果$x_0$是函数的第一类间断点，且函数在该点的单侧极限相等，但不等于函数在该点的值，或者函数在该点未定义，那么这样的间断称为可去间断点，$x_0$称为第一类可去间断点。

例如$f(x)=\frac{\sin(x)}{x}$：

由上述分析可知，点$x = 0$是该函数的第一类间断点，且$\lim_{x \to 0^+}\frac{\sin(x)}{x}=\lim_{x \to 0^-}\frac{\sin(x)}{x}=1\neq f(0)$，根据定义，点$x = 0$是第一类可去间断点。如果对函数$f(x)$进行如下补充定义：$f(x)=\begin{cases}\frac{\sin(x)}{x},&x\neq0\1,&x = 0\end{cases}$，那么根据定义，函数$f(x)$在点$x = 0$处将是连续的。

不可去间断点：如果$x_0$是函数的第一类间断点，且函数在该点的单侧极限不相等，那么这样的间断称为不可去间断点，$x_0$称为第一类不可去间断点。

例如：$f(x)=\begin{cases}x - 1,&x\leq0\x + 1,&x>0\end{cases}$。考虑在点$x = 0$处的单侧极限和函数值：$\left.\begin{array}{l} \exists \lim_{x \to 0^+}f(x)=1 \ \exists \lim_{x \to 0^-}f(x)= - 1 \ \exists f(0)= - 1 \end{array}\right} \Rightarrow$根据定义，点$x = 0$是第一类不可去间断点。

23. 函数可微的充要条件定理

25. 两个可微函数乘积的导数定理

使用列表中的定义61、64

如果函数$f(x)$和$g(x)$在点$x_0$处可微，那么函数$f(x)\cdot g(x)$在该点也可微，且$(f(x)\cdot g(x))’=f’(x)\cdot g(x)+f(x)\cdot g’(x)$。

证明：

由条件函数$f(x)$和$g(x)$在点$x_0$处可微，所以存在有限极限$\lim_{\Delta x \to 0}\frac{\Delta f}{\Delta x}=f’(x_0)$，$\lim_{\Delta x \to 0}\frac{\Delta g}{\Delta x}=g’(x_0)$。计算$(f(x)\cdot g(x))’$：

$$
\begin{align*}

(f(x)\cdot g(x))’&=\lim_{\Delta x \to 0}\frac{\Delta(f(x)\cdot g(x))}{\Delta x}\

&=\lim_{\Delta x \to 0}\frac{f(x+\Delta x)\cdot g(x+\Delta x)-f(x)\cdot g(x)}{\Delta x}\

&=\lim_{\Delta x \to 0}\frac{f(x+\Delta x)\cdot g(x+\Delta x)-f(x)\cdot g(x+\Delta x)+f(x)\cdot g(x+\Delta x)-f(x)\cdot g(x)}{\Delta x}\

&=\lim_{\Delta x \to 0}\left(\frac{f(x+\Delta x)-f(x)}{\Delta x}\cdot g(x+\Delta x)+f(x)\cdot\frac{g(x+\Delta x)-g(x)}{\Delta x}\right)\

&=\lim_{\Delta x \to 0}\left(\frac{\Delta f}{\Delta x}\cdot g(x+\Delta x)+f(x)\cdot\frac{\Delta g}{\Delta x}\right)\

&=\underbrace{\lim_{\Delta x \to 0}\frac{\Delta f}{\Delta x}}{f’(x_0)}\cdot\underbrace{\lim{\Delta x \to 0}g(x+\Delta x)}{g(x_0)}+\underbrace{\lim{\Delta x \to 0}f(x)}{f(x_0)}\cdot\underbrace{\lim{\Delta x \to 0}\frac{\Delta g}{\Delta x}}_{g’(x_0)}\

&=f’(x_0)\cdot g(x_0)+f(x_0)\cdot g’(x_0)

\end{align*}
$$

定理得证。

26. 两个可微函数商的导数定理

使用列表中的定义61、64

如果函数$f(x)$和$g(x)$在点$x_0$处可微，那么函数$\frac{f(x)}{g(x)}$在该点（当$g(x)\neq0$时）也可微，且$(\frac{f(x)}{g(x)})’=\frac{f’(x)\cdot g(x)-f(x)\cdot g’(x)}{g^{2}(x)}$。

证明：

$$
\begin{align*}

\left(\frac{f(x)}{g(x)}\right)’&=\lim_{\Delta x \to 0}\frac{\frac{\Delta f(x)}{g(x)}}{\Delta x}\

&=\lim_{\Delta x \to 0}\frac{\frac{f(x+\Delta x)g(x)-f(x)g(x+\Delta x)}{g(x)g(x+\Delta x)}}{\Delta x}\

&=\frac{\lim_{\Delta x \to 0}\frac{\Delta f}{\Delta x}\cdot g(x)-f(x)\cdot\lim_{\Delta x \to 0}\frac{\Delta g}{\Delta x}}{g^{2}(x)}\

\end{align*}
$$

30. 费马定理

  • 若$\Delta x > 0$，则$\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x} \leq 0$，根据极限的保号性，$\lim_{\Delta x \to 0^+}\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}=f_{+}’(x_0) \leq 0$。
  • 若$\Delta x < 0$，则$\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x} \geq 0$，根据极限的保号性，$\lim_{\Delta x \to 0^-}\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}=f_{-}’(x_0) \geq 0$。

综上可得$f’(x_0)=0$。

定理得证。

31. 罗尔定理

使用列表中的定义55、64、74、75

设函数$y = f(x)$满足：

  • 在闭区间$[a, b]$上连续；
  • 在开区间$(a, b)$内可微；
  • $f(a)=f(b)$。

那么在区间$(a, b)$内至少存在一点$x_0$，使得$f’(x_0)=0$。

证明：因为$y = f(x)$在$[a, b]$上连续，根据（魏尔斯特拉斯第二）定理，函数$y = f(x)$在闭区间$[a, b]$上能取到最大值和最小值，记$M = \max_{[a, b]}(f(x))$，$m = \min_{[a, b]}(f(x))$，则对于任意$x \in [a, b]$，有$m \leq f(x) \leq M$。

  • 若$m = M$，则对于任意$x \in [a, b]$，$m = M = f(x)$，即$f(x)$为常数函数，那么对于任意$x \in [a, b]$，$f^{\prime}(x) = 0$。
  • 若$m \neq M$：
    • 当$f(a)=f(b)=m$时，函数$y = f(x)$在$(a, b)$内取得最大值，即存在$a < x_0 < b$，使得$f(x_0)=M$。所以点$x_0$是局部极大值点。又因为$f(x)$在区间$(a, b)$内可微，所以$f(x)$在点$x_0$处可微。根据费马定理，$f’(x_0)=0$。
    • 当$f(a)=f(b)=M$时，函数$y = f(x)$在$(a, b)$内取得最小值，即存在$a < x_0 < b$，使得$f(x_0)=m$。所以点$x_0$是局部极小值点。又因为$f(x)$在区间$(a, b)$内可微，所以$f(x)$在点$x_0$处可微。根据费马定理，$f’(x_0)=0$。
    • 当$y = f(x)$在$(a, b)$内的最小值和最大值分别在不同点$x_0$和$x_1$处取得时，$x_0$和$x_1$是极值点。又因为$f(x)$在区间$(a, b)$内可微，所以$f(x)$在点$x_0$和$x_1$处可微。根据费马定理，$f’(x_0)=0$，$f’(x_1)=0$。

定理得证。

32. 拉格朗日定理

使用列表中的定义55、64

设函数$f(x)$满足：

  • 在闭区间$[a, b]$上连续；
  • 在开区间$(a, b)$内可微。

那么至少存在一点$c \in (a, b)$，使得$f(b)-f(a)=f’(c)\cdot(b - a)$。

证明：

考虑辅助函数$F(x)=f(x)-\frac{f(b)-f(a)}{b - a}\cdot(x - a)$。

因为$f(x)$具有这些性质，所以$F(x)$在闭区间$[a, b]$上连续，在开区间$(a, b)$内可微。

计算可得$\begin{cases}F(a)=f(a)-\frac{f(b)-f(a)}{b - a}\cdot(a - a)=f(a)\F(b)=f(b)-\frac{f(b)-f(a)}{b - a}\cdot(b - a)=f(a)\end{cases}$，即$F(a)=F(b)$。

因此，$F(x)$满足罗尔定理的所有条件，所以存在点$c \in (a, b)$，使得$F’(c)=f’(c)-\frac{f(b)-f(a)}{b - a}=0$，即$f’(c)=\frac{f(b)-f(a)}{b - a}$，进而$f’(c)\cdot(b - a)=f(b)-f(a)$

定理得证。

33. 柯西定理

使用列表中的定义55、64

设函数$f(x)$和$g(x)$满足：

  • 在闭区间$[a, b]$上连续；
  • 在开区间$(a, b)$内可微；
  • 对于任意$x \in (a, b)$，$g’(x) \neq 0$。

那么至少存在一点$c \in (a, b)$，使得$\frac{f(b)-f(a)}{g(b)-g(a)}=\frac{f’(c)}{g’(c)}$。

证明：

首先注意到$g(b)-g(a) \neq 0$，因为如果$g(b)-g(a)=0$，那么$g(a)=g(b)$，函数$g(x)$就满足罗尔定理的条件，根据罗尔定理，存在$c \in (a, b)$，使得$g’(c)=0$，这与定理的第三个条件矛盾。

引入辅助函数$F(x)=f(x)-\frac{f(b)-f(a)}{g(b)-g(a)}\cdot(g(x)-g(a))$。

因为$f(x)$和$g(x)$具有这些性质，所以$F(x)$在闭区间$[a, b]$上连续，在开区间$(a, b)$内可微。

计算可得$\begin{cases}F(a)=f(a)-\frac{f(b)-f(a)}{g(b)-g(a)}\cdot(g(a)-g(a))=f(a)\F(b)=f(b)-\frac{f(b)-f(a)}{g(b)-g(a)}\cdot(g(b)-g(a))=f(a)\end{cases}$，即$F(a)=F(b)$。

因此，$F(x)$满足罗尔定理的所有条件，所以存在点$c \in (a, b)$，使得$F’(c)=f’(c)-\frac{f(b)-f(a)}{g(b)-g(a)}\cdot g’(c)=0$，即$f’(c)-\frac{f(b)-f(a)}{g(b)-g(a)}\cdot g’(c)=0$，进一步得到$f’(c)=\frac{f(b)-f(a)}{g(b)-g(a)}\cdot g’(c)$，所以$\frac{f’(c)}{g’(c)}=\frac{f(b)-f(a)}{g(b)-g(a)}$。

定理得证。

34. 洛必达 - 伯努利定理（针对两个无穷小函数之比的极限）

使用列表中的定义35、36、64

设$f(x)$和$g(x)$满足：

  • 当$x \to x_0$时，$f(x)$和$g(x)$是无穷小函数或无穷大函数；
  • 在$x_0$的去心邻域$\stackrel{\circ}{U}(x_0)$内可微；
  • 在$\stackrel{\circ}{U}(x_0)$内$g’(x) \neq 0$；
  • 极限$\lim_{x \to x_0}\frac{f’(x)}{g’(x)}$存在。

那么极限$\lim_{x \to x_0}\frac{f(x)}{g(x)}=\lim_{x \to x_0}\frac{f’(x)}{g’(x)}$。

证明（针对无穷小函数的情况）：

由条件$f(x)$和$g(x)$当$x \to x_0$时是无穷小函数，

根据定义$\lim_{x \to x_0}f(x)=0 \Leftrightarrow \lim_{x \to x_0^+}f(x)=\lim_{x \to x_0^-}f(x)=0$，$\lim_{x \to x_0}g(x)=0 \Leftrightarrow \lim_{x \to x_0^+}g(x)=\lim_{x \to x_0^-}g(x)=0$。

由条件$\exists \lim_{x \to x_0}\frac{f’(x)}{g’(x)} \Leftrightarrow \lim_{x \to x_0^+}\frac{f’(x)}{g’(x)}=\lim_{x \to x_0^-}\frac{f’(x)}{g’(x)}=\lim_{x \to x_0}\frac{f’(x)}{g’(x)}$。

考虑$x \to x_0^+$时的极限（$x \to x_0^-$的情况证明类似），在点$x = x_0$处补充定义$f(x_0)=g(x_0)=0$，那么$f(x)$和$g(x)$在$U^{+}(x_0)$上有定义且连续。

考虑区间$[x_0, x]$，其中$x > x_0$。因为在$(x_0, x)$内$g’(x) \neq 0$，所以函数$f(x)$和$g(x)$满足柯西定理的条件，因此存在$c \in (x_0, x)$，使得$\frac{f(x)-f(x_0)}{g(x)-g(x_0)}=\frac{f’(c)}{g’(c)}$。

由于$f(x_0)=g(x_0)=0$，所以$\frac{f(x)}{g(x)}=\frac{f’(c)}{g’(c)}$。

因为$x_0 < c < x$，所以当$x \to x_0^+$时，$c \to x_0^+$。

这样就有$\lim_{x \to x_0^+}\frac{f(x)}{g(x)}=\lim_{x \to x_0^+}\frac{f’(c)}{g’(c)}=\lim_{c \to x_0^+}\frac{f’(c)}{g’(c)}=\lim_{x \to x_0^+}\frac{f’(x)}{g’(x)}$，同理$\lim_{x \to x_0^-}\frac{f(x)}{g(x)}=\lim_{x \to x_0^-}\frac{f’(x)}{g’(x)}$。所以$\exists \lim_{x \to x_0}\frac{f(x)}{g(x)}=\lim_{x \to x_0}\frac{f’(x)}{g’(x)}$。

定理得证。

35. 指数函数、幂函数和对数函数在无穷远处的增长速度比较

36. 带拉格朗日余项的泰勒公式推导：

设函数$f(x)$在点$x_0$的邻域$U(x_0)$内有定义，并且在该邻域内具有直到$(n + 1)$阶的导数。那么对于任意$x \in U(x_0)$，有等式：

$$
f(x)=\sum_{k = 0}^{n}\frac{f^{(k)}(x_0)}{k!}\cdot(x - x_0)^{k}+\frac{f^{(n + 1)}(x_0+\theta(x - x_0))}{(n + 1)!}\cdot(x - x_0)^{n + 1}
$$

其中$\theta \in(0,1)$ ，$\frac{f^{(n + 1)}(x_0+\theta(x - x_0))}{(n + 1)!}\cdot(x - x_0)^{n + 1}$是拉格朗日余项。

设$x \in U(x_0)$，为确定起见，设$x > x_0$。在区间$[x_0, x]$上考虑两个函数：

$$
\varphi(t)=f(x)-\sum_{k = 0}^{n}\frac{f^{(k)}(t)}{k!}\cdot(x - t)^{k}
$$

$$
\psi(t)=(x - t)^{n + 1}
$$

对于这两个函数有：

$$
\varphi(x)=f(x)-\sum_{k = 0}^{n}\frac{f^{(k)}(x)}{k!}\cdot(x - x)^{k}=f(x)-f(x)=0
$$

$$
\varphi(x_0)=f(x)-\sum_{k = 0}^{n}\frac{f^{(k)}(x_0)}{k!}\cdot(x - x_0)^{k}
$$

$$
\psi(x)=(x - x)^{n + 1}=0
$$

$$
\psi(x_0)=(x - x_0)^{n + 1}
$$

计算导数：

$$
\begin{align*}

\varphi’(t)&=\left(f(x)-f(t)-\sum_{k = 1}^{n}\frac{f^{(k)}(t)}{k!}\cdot(x - t)^{k}\right)’\

&=-f’(t)-\sum_{k = 1}^{n}\frac{1}{k!}\cdot\left(f^{(k + 1)}(t)\cdot(x - t)^{k}-k\cdot f^{(k)}(t)\cdot(x - t)^{k - 1}\right)\

&=-f’(t)-\sum_{k = 1}^{n}\frac{f^{(k + 1)}(t)}{k!}\cdot(x - t)^{k}+\sum_{k = 1}^{n}\frac{f^{(k)}(t)}{(k - 1)!}\cdot(x - t)^{k - 1}

\end{align*}
$$

在最后一个和式中引入新的求和指标$l = k - 1$，则$\sum_{k = 1}^{n}\frac{f^{(k)}(t)}{(k - 1)!}\cdot(x - t)^{k - 1}=\sum_{l = 0}^{n - 1}\frac{f^{(l + 1)}(t)}{l!}\cdot(x - t)^{l}=f’(t)+\sum_{l = 1}^{n - 1}\frac{f^{(l + 1)}(t)}{l!}\cdot(x - t)^{l}$。

所以$\varphi’(t)=-\frac{f^{(n + 1)}(t)}{n!}\cdot(x - t)^{n}$，另外$\psi’(t)=-(n + 1)\cdot(x - t)^{n}$，显然在区间$(x_0, x)$上$\psi’(t)\neq0$。

对区间$[x_0, x]$上的函数对$\psi(t)$和$\varphi(t)$应用柯西定理，有$\frac{\varphi(x_0)-\varphi(x)}{\psi(x_0)-\psi(x)}=\frac{\varphi(x_0)}{\psi(x_0)}=\frac{\varphi’(x_0+\theta(x - x_0))}{\psi’(x_0+\theta(x - x_0))}$，其中$\theta \in(0,1)$。

因为$0 < \theta < 1\Leftrightarrow 0 < \theta(x - x_0) < x - x_0\Leftrightarrow x_0 < x_0+\theta(x - x_0) < x$，令$c = x_0+\theta(x - x_0)\in(x_0, x)$。

结合前面的计算结果可得：

$$
\begin{align*}

\frac{\varphi(x_0)}{\psi(x_0)}&=\frac{f(x)-\sum_{k = 0}^{n}\frac{f^{(k)}(x_0)}{k!}\cdot(x - x_0)^{k}}{(x - x_0)^{n + 1}}\

&=-\frac{f^{(n + 1)}(x_0+\theta(x - x_0))}{n!}\cdot(x - x_0-\theta(x - x_0))^{n}\times\frac{1}{-(n + 1)(x - x_0-\theta(x - x_0))^{n}}\

&=\frac{f^{(n + 1)}(x_0+\theta(x - x_0))}{(n + 1)!}

\end{align*}
$$

即$f(x)-\sum_{k = 0}^{n}\frac{f^{(k)}(x_0)}{k!}\cdot(x - x_0)^{k}=\frac{f^{(n + 1)}(x_0+\theta(x - x_0))}{(n + 1)!}\cdot(x - x_0)^{n + 1}$。

上述推导是在$x > x_0$的情况下进行的，当$x < x_0$时推理类似；当$x = x_0$时，该定理结论显然成立。

定理得证。

37. 带佩亚诺余项的泰勒公式推导

  • 当$\Delta x>0$时，$f(x+\Delta x)\geq f(x)$，所以$f’(x)=f_{+}’(x)=\lim_{\Delta x \to 0^{+}}\frac{f(x+\Delta x)-f(x)}{\Delta x}\geq0$。

  • 当$\Delta x<0$时，$f(x)\geq f(x+\Delta x)$，所以$f’(x)=f_{-}’(x)=\lim_{\Delta x \to 0^{-}}\frac{f(x+\Delta x)-f(x)}{\Delta x}\geq0$。

综上，$\forall x \in(a, b)$，都有$f’(x)\geq0$ 。

证明（充分性）：

定理得证。

综上，$\forall x \in(a, b)$，都有$f’(x)\leq0$。

证明（充分性）：

由条件在区间$(a, b)$内所有$f(x)$可微的点处，不等式$f’(x)\leq0$都成立。设$x_1$和$x_2$是该区间内任意两点，且$a<x_1<x_2<b$。那么函数$y = f(x)$满足拉格朗日定理条件，所以$\exists c \in(x_1, x_2)$，使得$f(x_2)-f(x_1)=f’(c)(x_2 - x_1)$。因为$\forall x \in(a, b)$都有$f’(x)\leq0$且$x_2>x_1$，所以$f(x_2)-f(x_1)\leq0$，即$f(x_2)\leq f(x_1)$。因此，函数$y = f(x)$在$(a, b)$上单调不增。

定理得证。

45. 极值的第一充分条件（通过一阶导数判断）

使用列表中的定义54、64、76、77

设函数$y = f(x)$在$U_{\delta}(x_0)$内连续，在$\stackrel{\circ}{U}_{\delta}(x_0)$内可微。那么，如果$f’(x)$在经过点$x_0$时由负变正，则函数$f(x)$在该点有严格局部极小值；如果$f’(x)$在经过点$x_0$时由正变负，则函数$f(x)$在该点有严格局部极大值。如果$f’(x)$在点$x_0$的去心邻域内保持符号不变，则函数在该点没有极值。

证明：

考虑定理的第一个论断。如果在所有$x \in(x_0 - \delta, x_0)$上$f’(x)<0$，那么在半区间$(x_0 - \delta, x_0]$上函数$f(x)$递减，对于$x \in(x_0 - \delta, x_0)$，有$f(x)>f(x_0)$。在半区间$[x_0, x_0 + \delta)$上函数$f(x)$递增，对于所有$x \in(x_0, x_0 + \delta)$，有$f(x_0)<f(x)$。可见，$x_0$确实是严格局部极小值点。

定理的第二个论断证明类似。

在最后一种情况下，函数$f(x)$在区间$(x_0 - \delta, x_0 + \delta)$上要么递增，要么递减，这取决于$f’(x)$的符号 ，在这两种情况下，点$x_0$都不是极值点。

定理得证。

46. 极值的第二充分条件（通过二阶导数判断）

使用列表中的定义64、74、75、81

如果点$x = x_0$是函数$y = f(x)$的驻点，且函数$y = f(x)$在$x = x_0$处二阶可微，并且$f^{\prime\prime}(x_0)>0$（$f^{\prime\prime}(x_0)<0$），那么点$x = x_0$是函数的局部极小值点（局部极大值点）。

证明：

由条件$f^{\prime\prime}(x_0)>0$，可知$f’(x)$在$U(x_0)$内是递增的。因为$x_0$是函数$y = f(x)$的驻点，根据定义$f’(x_0)=0$。所以有$\begin{cases}f’(x)<0, & x<x_0\f’(x)>0, & x>x_0\end{cases}$，由极值的第一充分条件可知，$x_0$是局部极小值点。

当$f^{\prime\prime}(x_0)<0$时，证明过程类似。

定理得证。

47. 函数凸性的充分条件

使用列表中的定义64、86、87

设函数$f(x)$在区间$(a, b)$上二阶可微，并且在区间$(a, b)$内的每一点$x$处都满足不等式$f^{\prime\prime}(x)>0$。那么函数$f(x)$在该区间上是下凸的。如果在区间$(a, b)$内的所有点处二阶导数$f^{\prime\prime}(x)$都为负，那么函数$f(x)$在该区间上是上凸的。

证明：

这里只证明定理的第一个论断（第二个论断证明类似）。

考虑函数$y = f(x)$在点$(x_0, f(x_0))$（$x_0 \in(a, b)$）处的切线，其方程为$y = f(x_0)+f’(x_0)(x - x_0)$。

为确定起见，设$x_0<x<b$。那么切线在点$(x, f(x_0)+f’(x_0)(x - x_0))$和函数图像上点$(x, f(x))$的纵坐标之差为$\Delta y = f(x_0)-f(x)+f’(x_0)(x - x_0)$。

根据拉格朗日定理，$f(x)-f(x_0)=f’(c)(x - x_0)$，所以$\Delta y=(f’(x_0)-f’(c))\cdot(x - x_0)$，其中$c \in(x_0, x)$。

再应用一次拉格朗日定理可得$\Delta y=-f^{\prime\prime}(c_1)(c - x_0)(x - x_0)$，其中$c_1 \in(x_0, c)$。

由于$f^{\prime\prime}(c_1)>0$，$c - x_0>0$，$x - x_0>0$，所以$\Delta y<0$，即切线的点位于函数图像相应点的下方。

类似地，在$a<x<x_0$的情况下也可证明该论断。

因此，切线的点都位于函数图像相应点的下方，函数$f(x)$在区间$(a, b)$上是下凸的。

定理得证。

48. 拐点的必要条件

使用列表中的定义64、88

设函数$f(x)$在点$x_0$的邻域内二阶可微，并且二阶导数在该点连续。那么如果$x_0$是函数$y = f(x)$图像的拐点，则$f^{\prime\prime}(x_0)=0$。

证明：

假设$f^{\prime\prime}(x_0)\neq0$，为确定起见，设$f^{\prime\prime}(x_0)>0$。由于$f^{\prime\prime}(x)$在点$x_0$处连续，根据连续函数在点的邻域内保号定理，存在点$x_0$的邻域$U_{\delta}(x_0)$，使得在该邻域内所有点处$f^{\prime\prime}(x)>0$。那么在区间$(x_0 - \delta, x_0)$和$(x_0, x_0 + \delta)$上函数$f(x)$都是下凸的，这与在点$x_0$处存在拐点相矛盾。所以假设$f^{\prime\prime}(x_0)\neq0$不成立，即$f^{\prime\prime}(x_0)=0$。

定理得证。

49. 拐点的充分条件

使用列表中的定义50、88

设函数$f(x)$在点$x_0$的邻域$U_{\delta}(x_0)$内有定义，并且在该点连续。那么，如果在相应的去心邻域$\stackrel{\circ}{U}_{\delta}(x_0)$内函数$f(x)$有二阶导数，且二阶导数在经过点$x_0$时变号，则点$x_0$是函数$y = f(x)$的拐点。

证明：

为确定起见，设二阶导数$f^{\prime\prime}(x)$在$x \in(x_0 - \delta, x_0)$时为正，在$x \in(x_0, x_0 + \delta)$时为负。那么在$(x_0 - \delta, x_0)$上函数$f(x)$是下凸的，在$(x_0, x_0 + \delta)$上是上凸的，即经过点$x_0$时凸性方向改变。由此可知，$x_0$是函数$f(x)$的拐点。

定理得证。
